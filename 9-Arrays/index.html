Arrays (Vetores)

  • É uma estrutura do JavaScript para se trabalhar com todo tipo de lista
  • Seus elementos são identificados por um índice(index) inteiro começando pelo 0
  • Pode armazenar qualquer tipo de dado e ter qualquer tamanho
  • Pode ser criado e utilizado através de colchetes []
  • let array = []
  • let array = [1, 2, 4, 7]

  • array[0] = "Olá"
  • array[1] = 42
  • Possuem funções para vários casos de uso
    • Enfileirar e desenfileirar
    • Empilhar e desempilhar
    • Achar o índice(i) de um valor
    • Cortar e concatenar
    • etc
